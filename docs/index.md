# OpenGRC Framework Documentation Index

This directory contains the core architectural and governance documentation for the OpenGRC Framework.

The documentation supports structured modeling of RMF workflows, ATO artifacts, control mappings, Zero Trust alignment, and integration specifications.

### 3. ATO Artifact Lifecycle Model
Data-driven representation of:

- System Security Plan (SSP)
- Security Assessment Plan (SAP)
- Security Assessment Report (SAR)
- Plan of Action & Milestones (POA&M)
- Continuous Monitoring

- [ATO Artifacts Model](ato-artifacts.md)

---

### 4. Control Mapping Model
Vendor-neutral structured representation of NIST SP 800-53 controls, implementation tracking, evidence mapping, and telemetry linkage.

- [Control Mapping Model](control-mapping.md)

### 5. Zero Trust Alignment Model
Governance alignment across identity, device posture, network segmentation, applications, and data protection controls.

- [Zero Trust Alignment](zero-trust-alignment.md)
